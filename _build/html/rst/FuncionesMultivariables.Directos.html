FuncionesMultivariables.Directos.CaminataAleatoria module

FuncionesMultivariables.Directos.CaminataAleatoria.caminata_aleatoria(func, x0, paso, iteraciones=1000)[fuente]

Realiza una búsqueda de optimización usando el método de Caminata Aleatoria. La caminata aleatoria es un proceso donde el punto se mueve paso a paso en direcciones aleatorias dentro de un espacio, un plano o un espacio de mayor dimensión. Cada paso en este proceso es independiente de los anteriores.

Example:

>>> import numpy as np
>>> def func(x):
...     return np.sum(x**2)
>>> x0 = np.array([1.0, 1.0])
>>> paso = 0.1
>>> caminata_aleatoria(func, x0, paso)
array([0.0002154, 0.0005237])
Parámetros:

  • func (callable) – La función objetivo a minimizar.

  • x0 (ndarray) – El punto inicial de la búsqueda.

  • iteraciones (int, optional) – El número de iteraciones de la búsqueda. Default es 1000.

  • paso (float) – El tamaño del paso en cada iteración. Recomendacion de 0.1.

Devuelve:

El punto óptimo encontrado.

Tipo del valor devuelto:

ndarray

Raises:

TypeError

FuncionesMultivariables.Directos.HookeJeeves.hooke_jeeves(x0, deltas, alpha, epsilon, objective_function)[fuente]

Realiza la optimización usando el método de Hooke y Jeeves. El método de Hooke-Jeeves es un algoritmo de optimización directa utilizado para encontrar el mínimo de una función objetivo sin necesidad de derivadas.

Example:

>>> def objective_function(x):
...     return sum(xi**2 for xi in x)
>>> x0 = [0, 0]
>>> deltas = [1, 1]
>>> alpha = 2
>>> epsilon = 0.01
>>> hooke_jeeves(x0, deltas, alpha, epsilon, objective_function)
[0, 0]
Parámetros:

  • x0 (list) – El punto inicial de la búsqueda.

  • deltas (list) – Los tamaños de paso iniciales para cada dimensión.

  • alpha (float) – El factor de reducción para los tamaños de paso.

  • epsilon (float) – La tolerancia para la convergencia.

  • objective_function (callable) – La función objetivo a minimizar.

Devuelve:

El punto óptimo encontrado.

Tipo del valor devuelto:

list

FuncionesMultivariables.Directos.NelderMeadSimplex module

FuncionesMultivariables.Directos.NelderMeadSimplex.nelder_mead(f, x_start, gamma, beta, alpha, tol=1e-05, max_iter=1000)[fuente]

Implementa el método Nelder-Mead para la optimización de funciones sin derivadas. Este método es efectivo para optimización de funciones no lineales sin necesidad de derivadas, utilizando operaciones simples como reflexiones, expansiones y contracciones en un simplex que se adapta dinámicamente durante las iteraciones.

Example:

>>> def f(x):
...     return sum(xi**2 for xi in x)
>>> x_start = np.array([1.0, 1.0])
>>> gamma = 2.0
>>> beta = 0.5
>>> alpha = 1.0
>>> nelder_mead(f, x_start, gamma, beta, alpha)
array([0., 0.])
Parámetros:

  • f (callable) – La función objetivo a minimizar.

  • x_start (ndarray) – El punto inicial de la búsqueda.

  • gamma (float) – Parámetro gamma para la reflexión.

  • beta (float) – Parámetro beta para la contracción.

  • alpha (float) – Parámetro alpha para la expansión.

  • tol (float, optional) – Tolerancia de convergencia. Default es 1e-5.

  • max_iter (int, optional) – Número máximo de iteraciones. Default es 1000.

Devuelve:

El punto óptimo encontrado.

Tipo del valor devuelto:

ndarray
