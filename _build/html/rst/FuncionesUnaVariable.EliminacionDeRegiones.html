FuncionesUnaVariable.EliminacionDeRegiones.FibonacciSearch.fibonacci_search(funcion, epsilon, a, b)[fuente]¶

Implementación del método de búsqueda utilizando la secuencia de Fibonacci para minimización de funciones.

Usa todas las capacidades de Sphinx en esta descripción, por ejemplo, para dar ejemplos de uso …

Ejemplo:

>>> fibonacci_search(lambda x: x**2, 0.01, -1, 1)
0.0
Parámetros:

  • funcion (callable) – Función objetivo que se desea minimizar.

  • epsilon (float) – Precisión deseada para la aproximación del mínimo.

  • a (float) – Límite inferior del intervalo de búsqueda.

  • b (float) – Límite superior del intervalo de búsqueda.

Devuelve:

Punto donde se estima que se encuentra el mínimo de la función.

Tipo del valor devuelto:

float

Muestra:

ValueError – Si el intervalo no está correctamente definido.

FuncionesUnaVariable.EliminacionDeRegiones.IntervalHalving module¶

FuncionesUnaVariable.EliminacionDeRegiones.IntervalHalving.interval_method(function, epsilon, a, b)[fuente]¶

Implementación del método del intervalo para minimización de funciones.

Este método busca el mínimo de una función en un intervalo [a, b] dividiendo iterativamente el intervalo y evaluando la función en puntos clave.

Ejemplo:

>>> interval_method(lambda x: (x-5)**2, 0.01, 0, 10)
5.0
Parámetros:

  • function (callable) – Función objetivo que se desea minimizar.

  • epsilon (float) – Precisión deseada para la aproximación del mínimo.

  • a (float) – Límite inferior del intervalo.

  • b (float) – Límite superior del intervalo.

Devuelve:

Punto donde se estima que se encuentra el mínimo de la función dentro del intervalo [a, b].

Tipo del valor devuelto:

float

Muestra:

ValueError – Si el intervalo no está correctamente definido.
