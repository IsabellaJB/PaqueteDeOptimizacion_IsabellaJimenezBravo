FuncionesUnaVariable.BasadosEnLaDerivada.MetodoDeBiseccion module¶

FuncionesUnaVariable.BasadosEnLaDerivada.MetodoDeBiseccion.metodo_biseccion(x, funcion, epsilon)[fuente]¶

Encuentra una raíz de una función usando el método de bisección. El método de bisección es un algoritmo numérico para encontrar raíces de una función en un intervalo dado. Funciona dividiendo repetidamente el intervalo por la mitad y seleccionando el subintervalo donde la raíz probablemente se encuentra, hasta que el intervalo sea lo suficientemente pequeño o se alcance una precisión especificada.

Example:

>>> def funcion(x):
...     return x**3 - x - 2
>>> x = (-2, 2)
>>> epsilon = 1e-6
>>> metodo_biseccion(x, funcion, epsilon)
1.5213797092437744
Parámetros:

  • x (tuple) – Intervalo inicial [a, b] donde se busca la raíz.

  • funcion (callable) – Función objetivo.

  • epsilon (float) – Tolerancia para la convergencia del método.

Devuelve:

Aproximación de la raíz encontrada.

Tipo del valor devuelto:

float

FuncionesUnaVariable.BasadosEnLaDerivada.MetodoDeLaSecante module¶

FuncionesUnaVariable.BasadosEnLaDerivada.MetodoDeLaSecante.metodo_secante(x_inicial, funcion, epsilon, iter_max=100)[fuente]¶

Encuentra una raíz de una función no lineal usando el método de la secante. El método de la secante es un algoritmo numérico iterativo para encontrar las raíces de una función no lineal. A diferencia del método de Newton-Raphson, no requiere la evaluación de la derivada de la función en cada paso, haciendo el cálculo de la derivada menos crítico.

Example:

>>> def funcion(x):
...     return x**3 - x - 2
>>> x_inicial = (1.0, 2.0)
>>> epsilon = 1e-6
>>> metodo_secante(x_inicial, funcion, epsilon)
1.5213797068045676
Parámetros:

  • x_inicial (tuple) – Tupla con los dos puntos iniciales para comenzar el método.

  • funcion (callable) – Función objetivo.

  • epsilon (float) – Tolerancia para la convergencia del método.

  • iter_max (int, optional) – Número máximo de iteraciones permitidas. Default es 100.

Devuelve:

Aproximación de la raíz encontrada.

Tipo del valor devuelto:

float

Raises:

ValueError si el método no converge después de iter_max iteraciones.

FuncionesUnaVariable.BasadosEnLaDerivada.NewtonRaphson module¶

FuncionesUnaVariable.BasadosEnLaDerivada.NewtonRaphson.newton_raphson(x, funcion, epsilon)[fuente]¶

Implementa el método de Newton-Raphson para encontrar una raíz de la función dada. El método de Newton-Raphson es un algoritmo iterativo utilizado para encontrar numéricamente las raíces de una función mediante la aproximación sucesiva de un punto inicial hacia la raíz, basándose en la derivada de la función.

Example:

>>> def funcion(x):
...     return x**2 - 2
>>> x_inicial = [1.0, 2.0]
>>> epsilon = 1e-6
>>> newton_raphson(x_inicial, funcion, epsilon)
1.4142135623746899
Parámetros:

  • x (list of float) – Lista de puntos iniciales de búsqueda.

  • funcion (callable) – Función objetivo para la cual se busca la raíz.

  • epsilon (float) – Tolerancia de convergencia del método.

Devuelve:

Valor aproximado de la raíz de la función.

Tipo del valor devuelto:

float

Muestra:

ValueError – Si la longitud de x es cero o si no se alcanza la convergencia.
